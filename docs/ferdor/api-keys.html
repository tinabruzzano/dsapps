🔑 API Ferdor AI

Come ottengo una chiave API?
Puoi generare una chiave API per usare Ferdor AI per attivare un codice personalizzato per Ferdor AI.
Come imposto una API Key in Ferdor Browser?
Una volta generato, apri Ferdor Browser e su una nuova scheda digita nella barra URL fe://settings. Poi nel campo "Search Engine API Key (Ferdor AI)" cancella tutto e incolla la chiave.
Qual è il motore di ricerca di queste API Keys?
Visto che Ferdor AI Usa API, solitamente è DuckDuckGo, Google non ha API.

Genera API

Clicca "Genera API Key" per ottenere una chiave
