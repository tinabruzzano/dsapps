# Novità in Ferdor Browser, versione 3.0

## Introduzione
In Ferdor 3.0 Abbbiamo aggiunto nuove funzionalità, corretto bug e aggiunte funzionalità.

## Gli Addons: `fe://addons`
Se andate nella Pagina Ferdor `fe://addons`, ci sarannoi nuovissimi Addons!
Potrete estendere, personalizzare e fare molto altro con gli Addons!
La cosa bella: **Puoi Svilupparli TU**. Ecco i link per fare le vostre: \
👉 [Guida su come fare Addon](https://demenigeco.github.io/dsapps/developers/Ferdor/how-to-plu) \
👉 [Guida su come Programmare in E++](https://demenigeco.github.io/dsapps/developers/Ferdor/how-to-e++)

## Le critiche sistemate: ecco cosa abbiamo Pachato
- ⬇️ **I Downloads:** ora i downloads funzionano correttamente, in modo da scaricare i file dal web
- 💻 **Velocità all'apertura:** ora Ferdor si apre più velocemente e si carica più velocemente
- 🚩 **Flags Pachati:** Ora i flags funzionano meglio è su sistemi con poca memoria RAM (ne consumavano molta)

## Bugs: ecco cosa deve essere sistemato
- ⬇️ **Download una volta:** la finestra di download non si chiude automaticamente al primo download.
