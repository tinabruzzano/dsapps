## 📃 Template Privacy
Se vuoi usare un file Privacy nel tuo progetto, puoi partire dal nostro template:  

```text
[DomePrograms Privacy, Edizione 2]
[Copyright xxxx (c) Tutti i Diritti Riservati]

Attenzione: Questo template è solo un riassunto iniziale e deve essere integrato con un’informativa completa conforme alle normative applicabili. Per tutta la Privacy Policy, usate i link in basso.

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

1. Definizioni: "nomeprogramma", il Logo di "nomeprogramma" fa parte di "tuoteam". L’autore del progetto è responsabile della gestione dei dati e della conformità alle normative privacy applicabili.

2. Dati trattati: l'autore definisce quali dati sono trattati, ad esempio: dati personali, preferenze, ecc.

3. Normative: questo template è progettato per supportare la conformità alle normative GDPR (UE).

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Informativa completa:
ITA: https://demenigeco.github.io/dsapps/service/it/informativa-della-privacy.html
ENG: https://demenigeco.github.io/dsapps/service/en/privacy-policy.html
```

Con questo file Privacy **Garantisci**:
- [x] NO preoccupazioni della privacy
- [x] Certificato DomePrograms
- [x] Link Privacy multi-lingue
- [x] Chiarezza e semplicità

Ma **non garantisci**:
- [ ] Che tutti conoscano DomePrograms
- [ ] Che non sia sempre chiaro
- [ ] Che sia molto esplicito

Potete trovare tutte le edizioni qui: [Edizioni](https://github.com/DeMENIGECO/dsapps/blob/main/Privacy-versions.md)
