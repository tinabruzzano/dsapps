Fare Addon per Ferdor e Ferdor Beta

Questa FAQ è dedicata a sviluppatori di addons.

Ecco come Fare Addons per Ferdor e Ferdor Beta.

Procedimento

Fai la cartella

Crea una cartella del nome che vuoi, ad esempio my-extension. Consigliamo nomi memorabili e corti, perché sarà il Nome in codice dell'estensione

File INFORMATION

meta {author {"tuonome"}; prod {"La Mia Estensione"};}
build {prod.build {aa.numeroversione}}
superusr {enabled {False}}
setuptools {actived {False}}
File SETUP

file {
    dir:{th.thisuserwithferdor + "my-extension.plu" #Sostituisci 'my-extension' con il nome in codice    
    }
}

Impachettazione

Ora selezionate tutto il contenuto della cartella con CTRL-A (o su MacOS COMMAND-A) poi comprimi in ZIP e rinomina: 


nome-in-codice.plu
Invio

Cliccando Qui

  • Ricorda:
    • Allega il file fatto
    • Sostituisci "Nome"
  • Rispetta:
    • Non devi fare contenuti pericolosi
    • Se violi la regola l'Addon non verrà messo
  • Tieni a mente:
    • Potrebbe volerci qualche ora/giorno
